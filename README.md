# debate_time_meter

> A time-meter for debate.

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production for windwos
npm run build:win32


```

### instruction
```
# 配置校徽，辩手图片
首页右键修改配置

# 自定义赛制
首页右键自定义赛制

# 展示页面功能
+ 双击切换全屏/窗口
+ 快捷键z x c测试提示音
+ 回车进入下一个页面

# 计时页面
+ 双击切换全屏/窗口
+ 开始/暂停计时 p
+ 上/下一环节 Pageup/Pagedown
+ 切换发言人 1 2 3 4 q w e r
+ 是否显示功能键 n
+ 放大计时模式 m
+ 退出 esc

```
